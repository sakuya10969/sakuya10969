## 🧩 開発スタイル

### API First 開発

OpenAPIでAPI仕様を定義し、
Orvalを用いてフロントエンドの型・APIクライアントを自動生成することで、

* 型の不整合の排除
* フロント/バックエンド間の認識ズレの削減
* 開発スピードの向上

を実現しています。

---

### 仕様駆動開発 × AIエージェント

**開発プロセス自体を最適化するための手段としてAIを活用**しています。
